to_json/1

to_json(Str) -> any()

Parses a '\r\n' delimited string to JSON using the default conversion. The array id will be set to "data" and elements will be parsed as 'key'-'value' pairs, with key parsed from the second position and value from the first.

to_json/2

to_json(Str, X2) -> any()

Parses a '\r\n' delimited string to JSON. Id and args are passed as a tuple. The first element of the tuple is a string identifying the array and the second is a list of tuples specifying the element identifier and position.

to_record/2

to_record(Rec, Str) -> any()

Parses a \r\n delimited string to a key value record. Intended to populate Nitrogen combo/list boxes via the record syntax.
